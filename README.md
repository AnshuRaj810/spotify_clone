Spotify Clone (Frontend)

This project is a front-end clone of the Spotify web player. It is built to practice creating real-world user interfaces and improve frontend development skills.

Overview

The application recreates the basic layout of Spotify, including a sidebar, music section, and player controls. The focus of this project is on design, structure, and creating a smooth user experience using only frontend technologies.

Tech Stack
HTML5
CSS3
Features
Spotify-like layout and design
Sidebar with navigation options
Music player section with control layout
Clean and organized UI
Simple and easy to understand code structure
